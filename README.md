 # 🌍 Wanderlust

> **An Airbnb-inspired full-stack travel accommodation platform that allows users to discover, create, and manage property listings with secure authentication and cloud-based image storage.**


---

##  Features

-  Secure Authentication & Authorization
-  Create, Edit & Delete Property Listings
-  Cloudinary Image Upload & Management
-  Reviews & Ratings System
-  Search by Title, Location & Country
-  Category-Based Browsing
-  Fully Responsive Design
-  Deployed on Render with MongoDB Atlas

---

## 🛠️ Tech Stack

**Frontend:** HTML • CSS • JavaScript • Bootstrap • EJS

**Backend:** Node.js • Express.js

**Database:** MongoDB • Mongoose

**Authentication:** Passport.js • Express Session

**Cloud & Deployment:** Cloudinary • MongoDB Atlas • Render

---

##  Getting Started

```bash
git clone https://github.com/shanmukhkakarla2007-collab/Wonderlust.git

cd Wonderlust

npm install

npm start
```
